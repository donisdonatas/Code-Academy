Aš ir programavimas

  Sveiki, mano vardas Donatas. Programavimu susidėmėjau prieš 10 metų, kai sugalvojau susikurti pirmą asmeninį puslapį. Nenorėjau eiti lengviausiu keliu ir pasinaudoti kažkokiu jau egsituojančiu TVS (turinio valdymo sistema), kuri palengvina visą darbą, bet susiprogramuoti viską pats. Pačioje pradžioje pradėjau nuo HTML + CSS. Šis darbas leido susipažinti su pagrindinėmis HTML kodavimo taisyklėmis ir kaip taip taikyti CSS.
  Vėliau, adatptavau tą patį puslapį su PHP ir SQL, kad būtų galima jau kažkokius duomenis imti iš duomenų bazės.
  Su JavaScript susidūriau kai pradėjau domėtis Forex investavimu. Tada kilo noras ant MT4 platformos susiprogramuoti automatinius pirkimo ir pardavimo sandorių algoritmus. Vėliau norėjau pereiti prie MT5 platformos, bet pasirodė, kad ten pasikeitė programavimo kalba iš JavaScript į C++. Kurios sintaksė skyrėsi, tad iš naujo visko perprogramuoti ant C++ iš JavaScript neužsiėmiau.
  Taip pat JavaScript mokiausi programuoti su Google App Script. Kurio pgalba galima automatizuoti Google Sheet funkcijas ar pasigilinus daugiau būtų galima plėsti visų kitų aplikacijų funkcionalumą.
  Darbe daug tenka su Excel, tad norint automatizuoti Excel funkcijas ir išplėsti galimybes mokiausi programuoti su Visual Basic for Applications (VBA).

Kokias programavimo kalbas aš mokiausi:

  • HTML + CSS
  • PHP
  • SQL
  • JavaScript
  • Java
  • VBA
